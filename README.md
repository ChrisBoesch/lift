lift
===========

Medicine Management

## On a new machine

1. `gem install --pre sass`
2. `npm install -g bower grunt-cli`

## Setup

1. `npm install -g grunt-cli`
2. `npm install`
3. `bower install`
3. `grunt server`

## Development

1. `grunt server`
2.  Browser will open and starts serving on http://localhost:9000 (with live reload)

## Test

1. `grunt test`
